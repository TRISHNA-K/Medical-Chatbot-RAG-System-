# Medical Chatbot – RAG System

A full-stack Retrieval-Augmented Generation (RAG) application that enables users to interact with custom knowledge bases using natural language. 
The system combines semantic retrieval with Large Language Models (LLMs) to generate accurate, context-aware responses from uploaded documents.

 ### Features
- Retrieval-Augmented Generation (RAG)
- Semantic document retrieval using vector embeddings
- Multi-turn conversational question answering
- PDF document ingestion and preprocessing
- Embedding generation using Sentence Transformers
- Pinecone vector database integration
- LangChain orchestration for retrieval and generation
- Flask-based REST backend
- Responsive web interface
- Dockerized deployment
- AWS deployment support
